 Setup & Play  
1. Clone the repo:  
   ```bash  
   git clone https://github.com/your-username/cli-pacman.git  
   cd cli-pacman  
   ```  

2. Start the game:  
   ```bash  
   python main.py  
   ```  

---

 Controls  
- Movement: Arrow keys or WASD  
- Pause: `P`  
- Quit: `Q`  

---

 Customization  
- Adjust game speed in `config.py`.  
- Toggle music/sound effects in settings.
